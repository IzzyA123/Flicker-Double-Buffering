# Flicker – Double Buffering (WinForms)

This project demonstrates a flickering animation and its improvement using double buffering in C# Windows Forms.

## Features
- Moving square animated diagonally across the form
- Infinite looping with edge bouncing
- Noticeable flicker without buffering
- Reduced flicker using double buffering (back buffer to front buffer)

## Technologies
- C#
- Windows Forms
- System.Windows.Forms.Timer
- Double buffering via OptimizedDoubleBuffer

## How to Run
Open the solution in Visual Studio and run the project.
